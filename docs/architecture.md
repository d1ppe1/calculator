\## Описание модулей



\### calc.py

Содержит все математические функции и точку входа в программу.



\#### Функции:

\- add(a, b) - сложение

\- subtract(a, b) - вычитание

\- multiply(a, b) - умножение

\- divide(a, b) - деление



\#### Блок if \_\_name\_\_ == "\_\_main\_\_":

Используется для демонстрации работы калькулятора при прямом запуске файла. При импорте модуля в другой код этот блок не выполняется.
